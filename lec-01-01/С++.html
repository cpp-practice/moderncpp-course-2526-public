Кроссплатформенность

#include <iostream>

#if defined(_WIN32)
constexpr auto OS = "Windows";
#elif defined(__linux__)
constexpr auto OS = "Linux";
#elif defined(__APPLE__)
constexpr auto OS = "macOS";
#else
constexpr auto OS = "Unknown OS";
#endif

int main() {
    std::cout << "Hello from " << OS << "!\n";
}

Высокоуровневость и низкоуровневость

#include <iostream>
#include <string>

// Platform-specific includes
#if defined(_WIN32)
    #include <windows.h>
#else
    #include <unistd.h>
    #include <limits.h>
#endif

std::string getCurrentDirectory() {
    char buffer[MAX_PATH];
    #if defined(_WIN32)
        // Windows implementation
        GetCurrentDirectoryA(MAX_PATH, buffer); // from windes.h
    #else
        // Linux/macOS implementation
        getcwd(buffer, PATH_MAX);
    #endif
    return std::string(buffer);
}

int main() {
    std::cout << "Current directory: " << getCurrentDirectory() << std::endl;
    return 0;
}

Компилируемость

Компиляция - преобразование текста программы в машинный код

  • для каждой платформы отдельно

  • позволяет отловить некоторые ошибки

  • нет накладных расходов при выполнении программы

  • при изменении программы нужно компилировать снова

Preprocessing

Что происходит:

  • Включение заголовочных файлов

  • Раскрытие макросов

  • Условная компиляция

Compilation

  • Парсинг, Синтаксический анализ (на выходе AST)

  • Семантический анализ (Проверка типов, разрешение имен, допустимые операции)

  • Оптимизации (на базе AST\IR, инлайнинг функций, удаление недостижимого кода, оптимизация циклов)

  • Генерация кода ассемблера

Assembly

  • Преобразование в машинный код

  • В объектных файлах могут быть ссылки на символы в других объектных файлах

  • На выходе объектные файлы в формате: ELF(Linux/Unix), COFF/PE(Windows), Mach-O (macOS)

  • В объектом файле есть код, данные, таблица символов

Программа посложнее

// main.cpp
#include <iostream>
#include <cassert>

bool isGood(int n) { return n == 42; }

int not_init_global_variable;
int init_global_variable = 32;

int main() {
    assert(not_init_global_variable == 0);
    assert(init_global_variable == 32);

    int n = 32;
    for (int i = 0; i < n; ++i) {
        if (isGood(i)) {
            std::cout << i << " is good!" << std::endl;
        }
    }
    return 0;
}

Основные конструкции

Statements

(Утверждения)

Части программы, которые выполняются последовательно.

Пример:

int main() {
    int i = 0;       // statement 1
    std::cin >> i;   // statement 2
    return i;        // statement 3
}

Types of statements

  • declaration statements

  • expression statements

  • compound statements

  • selection statements

  • iteration statements

  • jump statements

Declaration statements

Необходимы, чтобы ввести новые символы (идентификаторы) в программе.

Пример:

int n = 1;
double a = 1, b = 2;

Expression statements

Syntax:

[expression] ;

  • Выражение — комбинация операторов и операндов (Click me)

  • Null statement — в случае отсутствия выражения. (Квадратные скобки в синтаксисе указывают на опциональность)

Пример:

n += 3;             // expression statement
std::cout << n;     // expression statement
;                   // null statement

Compound statements

Syntax:

{ [statements...] }

Последовательность утверждений, обернутых в фигурные скобки.

Пример:

int main() {
    {                   // start of compound statement
        int n = 10;     // ???
        std::cout << n; // expression statement
    }                   // end of compound statement
}

Selection statements (simplified)

Syntax (since c++17):

if ([init-statement] condition) statement [else statement]

  • statement — любое утверждение

  • condition:

    • expression statement, результат которого может быть приведен к типу bool (contextually converted to bool)

    • объявление переменной non-array типа с brace-or-equals инициализацией

if-statement examples

int get_somehow() { /* hidden implementation */ }
std::optional<int> compute_result() { return 1; }
int main() {
    int x = get_somehow();
    if (x < 5) { // expression statement
        std::cout << "x < 5" << std::endl;
    }

    if (int x = get_somehow()) { // equal-init
        std::cout << "x is not zero" << std::endl;
    }
    if (std::optional<int> value = compute_result()) {
        std::cout << *value;
    }

    if (int x{get_somehow()}) { // brace-init
        std::cout << "x is not zero" << std::endl;
    }
}

Iteration statements

while (condition) statement                                    (1)
do statement while (expression)                                (2)
for (init-statement [condition]; [expression]) statement       (3)
for ( for-range-decl : for-range-init ) statement              (4)

Iteration examples

Пример для (4):

    // while loop with a declaration condition
    char cstr[] = "Hello";
    int k = 0;
    while (char c = cstr[k++])
        std::cout << c;
    std::cout << '\n';

    int j = 2;
    do // compound statement is the loop body
    {
        j += 2;
        std::cout << j << ' ';
    }
    while (j < 9);
    std::cout << '\n';

Iteration examples

    for (int i = 0; i < 10; ++i)

    for (int i = 0, *p = &i; i < 9; i += 2)
        std::cout << i << ':' << *p << ' ';

    for (int n = 0; char c = cstr[n]; ++n) // condition may be declaration
        std::cout << c;

    // принцип работы range-based for loop разберем в следующих лекциях
